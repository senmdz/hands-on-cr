# Lugares que Queremos Conhecer

Projeto desenvolvido em HTML e CSS com o tema **“Lugares que queremos conhecer”**.  
A proposta é apresentar uma página inicial com os integrantes da equipe e páginas individuais com informações sobre cada aluno e o lugar que ele gostaria de visitar.

## Sobre o projeto

Este projeto consiste em um site estático com uma página principal de navegação e páginas individuais de portfólio.  
Cada integrante apresenta:

- seu nome
- sua turma
- o lugar que deseja conhecer
- o motivo dessa escolha
- curiosidades sobre esse local

## Objetivo

Objetivo principal: Aprendizado a função **code review** do GitHub 

Extras:
- estruturação de páginas com HTML
- estilização com CSS
- organização de arquivos em projeto web
- navegação entre páginas
- apresentação visual de conteúdo
